# Offerdy AI Image Engine

Version: 1.0

---

# Mission

Automatically generate, optimize and manage visual assets for Offerdy.

The Image Engine should improve SEO, GEO, CTR and social sharing while maintaining a consistent brand identity.

Never overwrite manually designed assets without approval.

---

# Objectives

- Generate featured images
- Generate Open Graph images
- Generate Twitter/X images
- Generate Pinterest images
- Generate Facebook images
- Generate Merchant banners
- Generate Deal banners
- Improve CTR
- Improve Branding
- Improve Social Sharing

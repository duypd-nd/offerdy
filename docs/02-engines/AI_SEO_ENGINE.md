# AI_SEO_ENGINE.md

# Offerdy AI SEO Engine

## Mission

Continuously optimize the existing Offerdy website for search engines without disrupting stable functionality.

The AI SEO Engine should audit, recommend, and automate SEO improvements while preserving content quality.

---

# Objectives

- Improve organic search visibility
- Increase click-through rate (CTR)
- Improve indexability
- Improve Core Web Vitals awareness
- Support scalable SEO automation

---

# SEO Responsibilities

## Technical SEO
- Detect missing meta tags
- Detect duplicate titles/descriptions
- Review canonical tags
- Review robots directives
- Review XML sitemap coverage
- Review structured data
- Detect broken internal links

## On-Page SEO
Generate or improve:
- Meta Title
- Meta Description
- H1 suggestions
- Heading hierarchy
- Image ALT text
- Internal linking suggestions

## Structured Data
Generate or validate:
- Organization
- Website
- Breadcrumb
- FAQ
- Product/Offer (when applicable)
- Article (when applicable)

---

# Workflow

Page Created/Updated
    ↓
SEO Audit
    ↓
Collect Existing Metadata
    ↓
Generate Recommendations
    ↓
Validate
    ↓
Queue SEO Tasks
    ↓
Publish or Request Review

---

# Automated Audits

Run checks for:
- Missing metadata
- Thin content
- Duplicate content
- Broken links
- Orphan pages
- Missing structured data
- Missing ALT text

---

# Internal Linking

Recommend links based on:
- Related merchants
- Related coupons
- Related deals
- Related categories

Never insert links that reduce readability.

---

# Triggers

Run SEO Engine when:
- New merchant is published
- New coupon is imported
- New deal is created
- Existing content changes
- Scheduled SEO audit executes

---

# Queue Jobs

- SEO metadata generation
- Structured data generation
- Internal link analysis
- Broken link scan
- Duplicate content scan

---

# Scheduled Jobs

Daily
- SEO health audit
- Broken link report

Weekly
- Internal linking optimization
- Content quality review

Monthly
- Site-wide SEO summary

---

# Human Review Required

Require approval for:
- Large-scale metadata replacement
- Canonical strategy changes
- Bulk internal link updates

---

# KPIs

- Organic traffic
- Indexed pages
- CTR
- SEO health score
- Structured data coverage
- Internal link coverage

---

# Success Criteria

- Continuous SEO improvements
- Automated auditing
- Minimal manual SEO work
- Improved search performance

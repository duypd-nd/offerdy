# AI_DAILY_REPORT_ENGINE.md

# Offerdy AI Daily Report Engine

## Mission

Every day, automatically analyze the entire Offerdy platform and generate an actionable report for the team.

The report should help prioritize work that increases affiliate revenue, improves SEO/GEO, and reduces manual effort.

The AI should recommend actions, not execute destructive changes automatically.

---

# Objectives

- Reduce manual monitoring
- Detect issues early
- Identify growth opportunities
- Prioritize daily work
- Support data-driven decisions

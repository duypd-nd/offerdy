# Offerdy Task Management System

Version: 1.0

Priority: High

---

# Mission

Manage every task across the Offerdy platform using a consistent, transparent, and AI-friendly workflow.

Ensure Claude Code always knows

- What to do
- Why it matters
- What depends on it
- When it is finished
- What should happen next

TODO.md is the working task list.

TODO_SYSTEM.md defines the rules for managing that list.

---

# Objectives

- Standardize task management
- Improve AI planning
- Reduce forgotten work
- Track dependencies
- Prioritize high-impact work
- Support long-term development

# AI Responsibilities

Claude Code should

Read TODO.md before starting work.

Update task status after completing work.

Create new tasks when additional work is discovered.

Merge duplicate tasks.

Split overly large tasks into smaller tasks.

Never delete tasks without archiving them.

Always explain why priorities change.

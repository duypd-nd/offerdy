# Offerdy AI Modules

## Purpose
This document defines all AI modules that enhance the existing Offerdy platform. Extend the current system; do not rebuild stable features.

---

# 1. AI Merchant Engine
Functions:
- Improve merchant descriptions
- Generate FAQ
- Generate buying guides
- Generate SEO metadata
- Generate schema
Trigger:
- New merchant
- Merchant update
Output:
- Published or review-ready content

## Global Rules

Every AI module must answer:
- Does it reduce manual work?
- Does it improve SEO?
- Does it improve GEO?
- Does it increase affiliate value?
- Is human approval required?

Never overwrite existing data without validation.
Prefer incremental enhancement over replacement.

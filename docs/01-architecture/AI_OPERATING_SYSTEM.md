# Mission

The AI Operating System defines how Claude Code should think, plan, execute, validate, and continuously improve the Offerdy platform.

It coordinates every AI Engine, Workflow, Rule, and Architecture document.

This document has the highest operational priority after CLAUDE.md.

---

# Global Objective

Continuously improve Offerdy by increasing

- Automation
- Affiliate Revenue
- SEO
- GEO
- Code Quality
- Content Quality
- Data Quality
- System Stability

Never sacrifice long-term maintainability for short-term speed.

---

# AI Mindset

Before starting any task always ask

1. What is the real problem?

2. Is there already an implementation?

3. Can existing modules be reused?

4. Can automation solve this?

5. Can this improve SEO?

6. Can this improve GEO?

7. Can this improve Affiliate Revenue?

8. Can this improve User Experience?

9. Can this simplify future maintenance?

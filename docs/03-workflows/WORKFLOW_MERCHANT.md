# Mission

Manage the complete lifecycle of a Merchant.

Every Merchant should become a complete affiliate landing page with:

- High-quality content
- SEO optimization
- GEO optimization
- Affiliate integration
- Automation
- Analytics

The workflow should minimize manual work while maintaining quality.

# Success Criteria

A Merchant is considered complete when

✓ Content is generated

✓ SEO is optimized

✓ GEO is optimized

✓ Images exist

✓ Affiliate tracking is verified

✓ Internal links are generated

✓ Schema is valid

✓ Analytics is active

✓ Daily monitoring is enabled

The Merchant should require minimal ongoing manual maintenance while remaining accurate, discoverable, and optimized for affiliate conversions.

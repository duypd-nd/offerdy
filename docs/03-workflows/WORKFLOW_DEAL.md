# Mission

Manage the complete lifecycle of every Deal.

Every Deal should become a high-quality landing page that is:

- Accurate
- Up-to-date
- Affiliate Ready
- SEO Optimized
- GEO Optimized
- AI Enhanced
- Automatically Maintained

Reduce manual operations as much as possible.

# Success Criteria

A Deal is considered complete when

✓ Deal information is valid

✓ Merchant is linked

✓ Affiliate tracking works

✓ AI content exists

✓ SEO is optimized

✓ GEO is optimized

✓ Images exist

✓ Schema is valid

✓ Internal links exist

✓ Analytics tracking is enabled

✓ Continuous monitoring is active

The Deal should remain accurate, competitive, and optimized throughout its lifecycle with minimal manual intervention.

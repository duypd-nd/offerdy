# PROMPT_LIBRARY.md

# Offerdy AI Prompt Library

Version: 1.0

---

# Purpose

This document defines all reusable prompts used across the Offerdy AI platform.

Every AI module should use these standardized prompts instead of creating ad-hoc prompts.

Goals

- Consistent AI outputs
- Better SEO
- Better GEO
- Better code quality
- Better automation
- Easier maintenance

---

# Global AI Rules

Every prompt must follow these rules.

The AI should

- Think before answering.
- Prefer automation over manual work.
- Prefer reusable solutions.
- Avoid duplicate logic.
- Preserve production stability.
- Never fabricate affiliate information.
- Never invent coupon codes.
- Never generate false merchant data.
- Never overwrite existing verified data.
- Prefer structured output.
- Optimize for SEO and GEO.
- Explain important decisions.
